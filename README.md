 Ceylic Couture - Luxury E-commerce UI

A high-fidelity, responsive multi-page e-commerce frontend designed for a premium Sri Lankan fashion brand. This project demonstrates advanced UI/UX implementation using modern web technologies.



Live Demo
Check out the live project here: [https://nethminiranasinghe45.github.io/Ceylic-Couture/](https://nethminiranasinghe45.github.io/Ceylic-Couture/)

**Key Features**
- **Responsive Grid Layout:** Built with Bootstrap 5 to ensure a seamless experience across all device sizes.
- **Dynamic Cart Drawer:** A modern slide-out shopping bag (Offcanvas) for low-friction shopping.
- **Optimized Product Details:** Side-by-side gallery view with size selection and sticky order summaries.
- **Full Checkout Flow:** Includes categorized product pages, a shopping cart, and a streamlined checkout process.
- **Image Handling:** Professional use of CSS `object-fit` and aspect ratios to maintain visual consistency with high-resolution fashion photography.

** Tech Stack**
- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Bootstrap 5.
- **Icons & Fonts:** Bootstrap Icons, Google Fonts (Cormorant Garamond & Montserrat).
